Danh sách mã ngôn ngữ (Language Code) trong HTML

Bài trước Bài kế tiếp

- Thông thường, việc khai báo ngôn ngữ của trang web sẽ giúp trình duyệt và các công cụ tìm kiếm hiểu rõ hơn về trang web của chúng ta, điều đó cũng có nghĩa là trình duyệt và các công cụ tìm kiếm sẽ chọn lựa sự hỗ trợ tốt nhất và phù hợp nhất dựa vào ngôn ngữ trang web của chúng ta.

- Để khai báo ngôn ngữ của trang web thì ta sử dụng thuộc tính lang và đặt nó bên trong thẻ <html> với cú pháp như sau:

<html lang="mã ngôn ngữ">

- Ví dụ: Trình duyệt và các công cụ tìm kiếm sẽ biết được trang web của đoạn mã bên dưới sử dụng ngôn ngữ Tiếng Việt.

<!DOCTYPE html>
<html lang="vi">
    <head>
        <title>Lập Trình Web</title>
    </head>
    <body>
        <p>Ngôn ngữ đánh dấu văn bản HTML</p>
        <p>Ngôn ngữ lập trình JavaScript</p>
        <p>Ngôn ngữ lập trình PHP</p>
    </body>
</html>
