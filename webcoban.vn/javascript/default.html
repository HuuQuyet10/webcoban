1) JavaScript là gì?

- JavaScript (viết tắt là JS) là một ngôn ngữ lập trình kịch bản phía máy khách, mã lệnh được thực thi bởi trình duyệt của người dùng.

- JavaScript được sử dụng rộng rãi trong việc kết hợp với HTML/CSS để thiết kế web động.

Chú thích: Web động là loại trang web mà những nội dung trên nó có thể tự động được cập nhật hoặc thay đổi bởi sự tương tác của người dùng. Trái ngược với web động, web tĩnh là loại trang web mà những nội dung trên nó luôn luôn cố định (không bao giờ thay đổi), điển hình là một trang HTML thuần.

- Dưới đây là một số ít ví dụ về ứng dụng của JavaScript trong việc thiết kế website:

  • Kiểm tra dữ liệu nhập vào từ biểu mẫu (form).
    • CẬP NHẬT THÔNG TIN CÁ NHÂN
    Họ tên:
    Ngày sinh:
    Giới tính: Nam Nữ
    THÔNG TIN CÁ NHÂN
    Họ tên: .......................
    Ngày sinh: .......................
    Giới tính: .......................
  • Xây dựng một cái đồng hồ thời gian thực.
  • Hiện vầ ẩn nội dung trên trang web.
  • Xây dựng một cái máy tính toán học đơn giản.

Java và JavaScript là hai ngôn ngữ hoàn toàn khác nhau

2) Tầm quan trọng của JavaScript

- JavaScript là một ngôn ngữ lập trình phổ biến bậc nhất thế giới và là một trong ba ngôn ngữ không thể thiếu đối với một lập trình viên web.

  • HTML
  • CSS
  • JavaScript

- Cũng như các bạn đã thấy qua những ví dụ phía trên, JavaScript đã mang đến cho chúng ta những gì. Nếu một trang web mà không có JavaScript thì trang web đó thật sự là rất nhàm chán và vô vị.

3) Gợi ý cách học JavaScript hiệu quả

- Trước tiên, để có thể học và tiếp thu tốt được JavaScript thì yêu cầu các bạn đọc cần phải có kiến thức cơ bản về HTML và CSS.

- JavaScript đã được ra đời từ rất lâu (tháng 5, 1995) với khối lượng kiến thức khổng lồ đan xen lẫn nhau và tương đối rời rạc. Do đó, một người mới bắt đầu tìm hiểu về JavaScript sẽ khó mà biết được phải tìm hiểu từ đâu và theo trình tự nào là đúng.

- Tuy nhiên, với niềm đam mê và sự nổ lực hết mình, chúng tôi đã biên soạn thành công loạt bài hướng dẫn học JavaScript theo một trình tự mạch lạc từ cơ bản đến nâng cao. Bài học trước cung cấp kiến thức cho bài học sau, giúp các bạn đọc dễ dàng tiếp thu được nội dung của bài viết.
